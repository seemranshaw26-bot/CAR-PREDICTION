# 🚗 Car Price Prediction

A Machine Learning web application that predicts the selling price of a used car based on features such as year, present price, kilometers driven, fuel type, 
seller type, transmission type, and ownership history.

## Features

* Predicts used car selling prices
* Interactive web interface built with Streamlit
* Data preprocessing and feature engineering
* Machine Learning model trained using Scikit-Learn
* Simple and user-friendly design

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-Learn
* Streamlit
* Matplotlib

## Dataset Features

* Car Name
* Year
* Present Price
* Kilometers Driven
* Fuel Type
* Seller Type
* Transmission
* Owner

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/CAR-PREDICTION.git
```

Navigate to the project folder:

```bash
cd CAR-PREDICTION
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
